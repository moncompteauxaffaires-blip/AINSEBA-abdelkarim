## 📖 Description du Jeu

Le prince doit traverser **7 royaumes** pour sauver sa princesse captive par un boss puissant.

### 🎯 Objectif
- Combattre les boss des 7 royaumes
- Conserver ses 3 vies pour compléter l'aventure
- Vaincre le boss final pour libérer la princesse

### ⚔️ Gameplay
- Combat **au tour par tour**
- Le prince attaque avec son épée
- Le boss riposte à chaque attaque
- Les dégâts du prince : 1-5
- Les dégâts du boss : 1-3
- Chaque boss éliminé = progression au royaume suivant

### 🏆 Condition de Victoire
Vaincre tous les boss des 7 royaumes pour sauver la princesse !

Bonne chance, prince ! 👑✨
